# SAP-PP-Automobile-Project

## Description
Implementation of SAP S/4HANA PP module for an automobile manufacturing company 
demonstrating a complete Plan-to-Produce cycle.

## Modules Used
- SAP PP (Production Planning)
- SAP MM (Material Management)

## Key Transactions
MM01 | CS01 | CR01 | CA01 | MD01 | MD04 | CO01 | CO02 | MIGO | MB31

## Process Flow
Material Creation → BOM → Work Center → Routing → MRP → Production Order → Release → Goods Issue → Goods Receipt
